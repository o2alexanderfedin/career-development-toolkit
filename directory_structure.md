# Recommended Directory Structure

```
/Jobs/AI/
│
├── CLAUDE.md                    # Working rules and guidelines
│
├── linkedin/                    # LinkedIn content development
│   ├── posts/                   # LinkedIn posts
│   │   ├── drafts/              # Working drafts
│   │   ├── final/               # Final approved versions
│   │   └── archive/             # Historical posts 
│   │
│   ├── profile/                 # LinkedIn profile content
│   │   ├── headline.txt         # Professional headline
│   │   ├── summary.txt          # About section
│   │   └── experience/          # Experience descriptions
│   │       ├── company1.txt     
│   │       └── company2.txt
│   │
│   └── engagement/              # Engagement content
│       ├── comments.txt         # Template responses
│       └── messages.txt         # Outreach templates
│
├── resume/                      # Resume content
│   ├── versions/                # Different resume versions
│   │   ├── technical.txt        # Technical focus
│   │   └── executive.txt        # Leadership focus
│   │
│   ├── sections/                # Resume components
│   │   ├── skills.txt           # Skills section
│   │   ├── experience.txt       # Experience section
│   │   └── education.txt        # Education section
│   │
│   └── customized/              # Company-specific resumes
│
├── job_search/                  # Job search materials
│   ├── target_roles.txt         # List of target positions
│   ├── target_companies.txt     # List of target companies
│   ├── cover_letters/           # Cover letter templates 
│   └── interview_prep/          # Interview preparation
│       ├── questions.txt        # Common questions
│       ├── answers.txt          # Prepared answers
│       └── stories.txt          # Achievement stories
│
└── tests/                       # Content validation tests
    ├── criteria.txt             # Test criteria
    ├── personas/                # Reviewer personas
    │   ├── hiring_manager.txt   # Technical hiring manager 
    │   └── recruiter.txt        # Non-technical recruiter
    └── reports/                 # Test results
```
